## Getting Started

### Setup

To setup the project locally, clone the repository using:

```
$ git clone https://github.com/TahenyBELGUITH/Touha_budget_app.git
```

### Install dependencies
run:
```
$ bundle install
```

### Setup the database
run:
```
$ rails db:create
$ rails db:migrate
$ rails db:seed
```

### Run the app
run:
```
$ rails s
```

### Run the tests
run:
```
$ rspec
```
